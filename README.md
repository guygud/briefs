# Бриф на геймификацию

Интерактивный HTML-шаблон для брифинга на геймификацию. Работает без бэкенда.

## Как использовать

1. Откройте `gamification-brief.html` в браузере (локально или на GitHub Pages)
2. Заполните форму
3. Нажмите **«Поделиться брифом»** — ссылка с данными скопируется в буфер
4. Или **«Скачать JSON»** — сохраните данные в файл
5. **«Печать / PDF»** — печать через браузер (Cmd+P → Сохранить как PDF)

## GitHub Pages

Репозиторий уже настроен для GitHub Pages. Включите в настройках:

**Settings → Pages → Source:** Deploy from a branch  
**Branch:** main (или master)  
**Folder:** / (root)

Страница будет доступна по адресу: `https://guygud.github.io/briefs/gamification-brief.html`
